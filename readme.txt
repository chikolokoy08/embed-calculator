== Description ==

Embed various type of calculators or converter. From weight, height, money, units and simple calculator in a breeze of a shortcodes.

This plugin provides ease to choose a variety of calculator or converter for your sites need. For the initial version we provide calculator / converter for currency, measurement, mass and custom units.

Supported units conversion:

*   Basic Calculator
*	Measurement
*	Mass
*   Custom: Range Converter

For ver1.1
- Will support currency conversion

= Shortcodes =
[Check documentation here](http://jumpstartcreatives.com/plugins/embed-calculator-plugin)

== Installation ==

1. Upload the unzipped folder embed-calculator to the /wp-content/plugins/directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings/Embed Calculator' menu in Wordpress to setup your global seetings for this plugin.

== Frequently Asked Questions ==

= Does the embed calculator can be edited or modified? =

Yes. It's either through the Global Setup or from the shortcodes' custom attributes.

= Does this plugin has conflict to the embedding system of Wordpress? =

Nope. The plugin provides a non conflict implementation to avoid errors from Wordpress core embedding function.

= Do you have detailed documentation and demos? =

Yes. You can visit [here](http://jumpstartcreatives.com/plugins/embed-calculator-plugin)
